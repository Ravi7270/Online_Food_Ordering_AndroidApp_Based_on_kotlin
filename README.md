# # Online Food Ordering App (Kotlin)

This is a simple Android application for ordering food, developed using Kotlin.
The app allows users to browse menu items, add them to cart, and place orders.

---

## Features

* View food menu
* Search items
* Add to cart
* Order food
* View order history
* User login and signup
* Profile management

---

## Tech Stack

* Kotlin
* XML (UI Design)
* RecyclerView
* Activities and Fragments
